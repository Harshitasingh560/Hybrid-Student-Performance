# Hybrid Intelligent System (Fuzzy + Neural Network)

## Objective
To predict student performance using fuzzy logic and neural networks.

## Inputs
- Attendance
- Assignment Marks
- Test Marks

## Output
- Performance (Poor, Average, Good)

## Method
- Fuzzy Logic for rule-based decision
- Neural Network for learning

## Hybrid Approach
Fuzzy system provides initial output, neural network improves accuracy.

## Tools Used
- MATLAB Fuzzy Logic Toolbox
- Neural Network Toolbox
